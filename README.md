# 🐾 Pet Shop E-commerce Website

A modern and responsive pet shop e-commerce website built with React and Tailwind CSS.

This project provides a clean shopping experience for pet products with modern UI design, reusable components and responsive layouts for all devices.

## ⚙️ Installation

Clone the repository:

```bash
git clone YOUR_REPOSITORY_LINK
```

Go to the project folder:

```bash
cd pet-shop
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
